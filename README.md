## Features

- Video Seeking UI interaction
  - when press  ←, → key 
- Play/Pause UI interaction
  - when press spacebar
- Volume UP/DOWN UI interaction
  - when press ↑, ↓ key
- Play rate UI interaction
  - when press > (Shift + .), < (Shift + ,)
